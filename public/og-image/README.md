# Open Graph Preview Image

Add your main OG (Open Graph) image that will be displayed when sharing your portfolio on social media:

- `og-image.jpg` - Default OG image (1200x630px)

**Recommended dimensions:** 1200x630px (1.91:1 aspect ratio)

The OG image is referenced in `src/data/config.ts` and used in `app/layout.tsx` for social media previews.
